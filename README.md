# ✈️ Travel Planner App

An AI-powered travel planning web application that takes your trip preferences 
and returns personalized flight options, hotel recommendations, and top places 
to visit at your destination.

## Features
- Smart travel form — destination, duration, budget, travel reason
- Live flight search via real travel APIs
- Hotel recommendations filtered by budget and rating
- Top places to visit near the destination
- AI-generated trip summaries and recommendations
- Real-time weather at destination
- Comparison of options across multiple sources

## Tech Stack
- **Frontend** — React + Tailwind CSS
- **Backend** — Python + FastAPI
- **AI Layer** — Claude API + scikit-learn
- **APIs** — Aviationstack, Google Places, Open-Meteo
- **Deploy** — Vercel (frontend) + Railway (backend)
